<h1>Windows Service Barcode Reader & Writer Demo</h1>

Demo application shows how to read and write barcodes in Windows service application. Demo uses functionality of <a href="https://www.vintasoft.com/vsbarcode-dotnet-index.html">VintaSoft Barcode .NET SDK</a>. Source codes for C# and VB.NET are available.
